Positions

Static

Por defecto, no se aplica ninguna propiedad de posición. Utiliza el flow normal del documento.

Static

Relative

Afecta la posición top right bottom left, y conserva su espacio

Relative

Absolute

Afectan la posición top right bottom left
No ocupa su espacio original
Se posicion con respecto al padre mas cercano que tenga una propiedad position diferente de STATIC(normalmente relative). Si no la encuentra, utiliza body.

Absolute

Fixed

Afecta las position, no ocupa espacio, se posiciona respecto al navegador(viewport), se mantiene fijo en la pantalla(no se nueve incluso con scroll)

Fixed

Sticky

Comportamiento combiando que puede comportarse como fixed y relative.
Utiliza el top y left como punto de adhesión.
Se posiciona respecto a la ventana del navegador(viewport)
Se queda fijo en la pantalla(fixed) hasta que el elemento padre se acaba.

Sticky
